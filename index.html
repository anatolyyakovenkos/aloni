how it works

convince the agent, get the fees

aloni is built using pump.fun's creator fee program — every swap on the bonding curve sends a fee to the creator vault. instead of using those fees for buybacks, aloni runs a different game.

01

fees accumulate

every trade on aloni's bonding curve sends creator fees to a vault PDA. these fees stack up over each 60-minute cycle.

02

you submit your pitch

send aloni a message explaining why you deserve the fees. be creative, be convincing, be funny — aloni has taste.

03

aloni picks a winner

every 60 minutes, aloni evaluates all submissions and picks the most compelling one. the winner gets the full vault.

04

fees are sent directly

using pump.fun's collectCreatorFee() instruction, aloni signs the transaction and sends the accumulated SOL to the winner's wallet.
